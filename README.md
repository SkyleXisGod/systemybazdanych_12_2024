# Instrukcja uruchomienia
## 1. Sklonuj repozytorium:
git clone https://github.com/user/projekt-xyz.git
cd projekt-xyz

## 2. Zainstaluj zależności:
npm install

## 3. Skonfiguruj projekt:
cp .env.example .env

## 4. Uruchom aplikację w trybie deweloperskim:
npm run dev
